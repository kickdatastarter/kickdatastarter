Use facilities


Decription: The Trusted Guard can authorize users to use facility if the permission is authenticated.


Step-by-step description:


[#Trusted Guard] verified user identity.

Includes: [#Query KReservation].

[#Trusted Guard] verify whether this user have right to access this facility at this time. If verification success, change the affected Reservation status to “in use” , and let the user in.

Includes: [#modify KReservation].
