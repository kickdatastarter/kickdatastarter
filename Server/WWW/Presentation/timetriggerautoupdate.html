Time triggered auto update


Decription: A program that checks whether a reservation needs to be updated


Step-by-step description:


[#LFRS] triggers the program every 15min.

Includes: [#Query KReservation]. [#Reservation Tracker] checks each reservation which just expired the 15min "grace period", but not "in use".

Includes: [#Modify KReservation]. [#Reservation Tracker] sets affected Reservation status to "First in First Serve".
